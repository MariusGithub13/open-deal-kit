# III · The Deal Team

**20 agents, one for every job in a deal, from sourcing to close.**

An agent is a role you set up once and reuse. Each folder here is an installable Claude skill (`SKILL.md`) with a complete copy-ready prompt, the role, the rules, and the output format already written. Install it as a skill and just describe your deal, or paste the prompt straight into Claude.

The agents are ordered to follow a real deal. Feed the output of one into the next, and run the **[Citation Checker](citation-checker/SKILL.md)** over any brief before you rely on it.

### I · Sourcing
| # | Agent | Does |
|---|-------|------|
| 01 | [Deal Scout](deal-scout/SKILL.md) | Builds a focused list of acquisition targets that fit your thesis |
| 02 | [Thesis Builder](thesis-builder/SKILL.md) | Sharpens a vague "I want to buy a business" into a precise, testable thesis |
| 03 | [Outreach Writer](outreach-writer/SKILL.md) | Owner-direct outreach that gets replies without sounding like a broker |
| 04 | [Teaser Requester](teaser-requester/SKILL.md) | The note that gets an owner or broker to share the teaser or financials |

### II · Screening
| # | Agent | Does |
